Logistics Application Design - Python

My first Python application was a logistics management system, designed to display information from spreadsheets whilst allowing other users access to the sheets. It was conceived to solve the issue of having to open up a spreadsheet, to enter or display specific information when this would prevent access by other users. This application centralises all the key information and only accesses the file briefly upon save, thus maintaining access. It was based on existing ERP software and designed to allow full CRUD functionality.
